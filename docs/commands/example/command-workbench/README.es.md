# Command Workbench

Este ejemplo separa commands compartidos del repositorio y ergonomía personal.

- Copia `project/.claude/commands/` dentro de la raíz de tu repositorio.
- Copia `project/.claude/settings.json` y `project/.claude/statusline.sh` dentro de esa misma `.claude/` de proyecto.
- Copia `user-home/.claude/keybindings.json` en `~/.claude/keybindings.json`.

Después de copiar los archivos, vuelve a abrir Claude Code o ejecuta el workflow de settings correspondiente para que el runtime recoja los cambios.
