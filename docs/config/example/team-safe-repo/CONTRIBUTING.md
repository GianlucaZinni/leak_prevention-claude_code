# Contributing

## AI Assistance Disclosure

If you use any AI tools to help with your contribution, disclose that in your pull request description.

### What to Disclose

| AI Usage | Example Disclosure |
| --- | --- |
| AI-generated code | "This PR was written primarily with Claude Code." |
| AI-assisted research | "I used ChatGPT to understand the codebase." |
| AI-suggested approach | "Copilot suggested the algorithm structure." |
| AI-drafted docs | "Documentation was drafted with Claude assistance." |

### What Does Not Need Disclosure

- Trivial autocomplete for a single keyword or short phrase
- IDE syntax helpers such as formatting or auto-imports
- Grammar and spell checking
- Code formatting tools

### Suggested Format

```markdown
## AI Assistance

This PR was developed with assistance from [Tool Name].
Specifically:
- [What AI helped with]
- [What you did manually]

All code has been reviewed and understood by the author.
```
